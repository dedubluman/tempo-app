## Rate Limit Logic

Server mirrors: app/api/sponsor/route.ts → 10 req/60s per IP
Client mirrors: rateLimiter = createRateLimiter({ maxRequests: 10, windowMs: 60_000 })

Thresholds:
- remaining <= 2 → "warn" status (shows "Approaching rate limit")
- remaining === 0 → "limited" status (shows "Rate limit reached. Pausing for Xs.")
- now - lastPaymentAt < 5000 → "throttled" (shows "Throttled (min 5s interval)")
